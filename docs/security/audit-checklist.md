# 보안 감사 체크리스트

## 범위
- 컨센서스, P2P, 지갑/키 관리, 직렬화/파서

## 컨센서스
- 블록/트랜잭션 검증 누락 여부
- 난이도/타임스탬프 경계 조건
- 재조직 시 UTXO 롤백 정확성
- 수수료/보상 상한 검증

## 네트워크
- 메시지 길이/리스트 크기 제한
- 레이트 리밋/밴 정책
- 잘못된 메시지 처리 경로(패닉 방지)
- 핸드셰이크/버전 호환성

## 지갑/키
- 키 생성/서명 검증
- 지갑 파일 암호화/복호화 검증
- 비밀번호/키 파생 파라미터 점검

## 직렬화/파서
- JSON/binary 경계 입력
- 오버플로/언더플로
- canonicalization 불일치 여부

## 운영
- 백업/복구 경로 검증
- 마이그레이션 안전성
- 로그/모니터링 정보 노출 여부
